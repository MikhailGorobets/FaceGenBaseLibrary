Use

Set up the path ('>' windows. '$' osx,ubuntu) for 64bit release version (VS2012 for Windows):

> PATH=%PATH%;~fgbl\bin\win\vs12\64\release
$ PATH=$PATH:~fgbl/bin/osx/gcc/64/release
$ PATH=$PATH:~fgbl/bin/ubuntu/gcc/64/release

View all available commands:

$> fgbl

Run all automated tests:

$> fgbl test all

View all manual (interactive) test categories (GUI currently only supported on Windows):

$> fgbl testm

Rebuild makefiles, solution files and project files by scanning existing source code files:

~fgbl$> fgbl cons

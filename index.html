Install

FaceGen executables load data from ~sdk/data/... This directory must be present in the directory containing the executable, or any of its parent directories. It must contain the file /data/_facegen_data_dir.flag

Example path setup:

Windows

> PATH=%PATH%;~sdk\bin\win\x64\vs17\release\

Ubuntu

$ PATH=$PATH:~sdk/bin/ubuntu/x64/clang/release/

MacOS

$ PATH=$PATH:~sdk/bin/macos/x64/clang/release/

Where ~sdk is the path in which you have placed this SDK.
